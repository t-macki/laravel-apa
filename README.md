laravel-apa
================

[Amazon Product Advertising API](https://affiliate.amazon.co.jp/assoc_credentials/home) APIをLaravelから利用するためのライブラリです。

## 動くLaravelのバージョン
- Laravel5.5以上



## Install

composer.jsonに以下の記述を加えてください。

```
{
  "require": {
    "macki/laravel-apa": "0.2.*@dev"
  }
}
```
